[share] 各种操作系统镜像

我提供了不同操作系统以及版本的iso镜像
存储在网站上的镜像都会用torrent包装
需要用Qbittorent来下载torrent
Windows下载：点击这里直接下载最新版本
Linux下载：在终端输入以下命令：

sudo apt install qbittorrent

要运行程序，输入：

qbittorrent
