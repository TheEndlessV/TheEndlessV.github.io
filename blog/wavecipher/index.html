[PROJ] Self-Created Line Language - WaveCipher

Language that can turn a line of string into a special line type.

Example:

Input:

hi! nice to meet you!

Output:

          _                  _           _                           _        _      
/\   _/\ / \   _ _ _   _/\ _/ \ /\ _    / \ _    _ /\ /\    _ _  /\ / \   /\ / \   _ 
  \/|   |   \_/ | | \/|   |    |  | |\/|   | |\_/ |  |  |\/| | \/  |   |\/  |   \_/ |
