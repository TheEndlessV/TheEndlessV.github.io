功能

我会在网站上发布博客，博客有不同的类型标签
常见的标签有：

  • 分享资源 (Share)
    该标签的博客存放资源的分享并对其介绍，会提及到不同的下载方式
    因为github的20MB限制，部分资源文件会被压缩或者是BT种子包装（即torrent文件）
    部分文件可能会被加密（AES-256加密，可不要尝试破解），密码只会被告知给部分人
  • 笔记 (Note)
    记录一些使用工具时的笔记，也会提到一些避坑的地方
  • 修复指南 (Fix)
    专讲修复，修复的可以是系统，或者使用工具时出的问题

使用建议

要想使用网站的大部分资源，需要有下载 BT文件 和 解压 的工具
这里colder推荐Qbittorent作为下载BT文件的工具

对于Windows用户，点击这里自动下载最新exe版本
对于Linux用户，输入以下命令安装Qbittorent:

sudo apt install qbittorrent

要运行它，只需输入以下命令：

qbittorrent

初次使用点 Agree 就可以了

在Windows里的解压，colder强烈推荐Bandizip，点击这里直接下载最新版本
对于Linux，直接用unzip命令就可以了，不懂的自行查
